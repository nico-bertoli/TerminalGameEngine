# TERME (Terminal Engine)

An engine for creating 2D terminal rendered games, built without any graphics libraries.<br>
This repo just contains the engine, to get some engine usages example you can checkout the [Terme Examples Repo](https://github.com/nico-bertoli/terme_examples)

#
Main features:
- Cross platform (Linux/Windows)
- Framerate-independent simulation (delta-time based)
- Collision system (OnCollisionEnter/Exit)
- Sorting order system
- Particle system
- Backgrounds support
